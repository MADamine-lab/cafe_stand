# Café Elbey — مقهى الباي

Site vitrine React (Vite) pour le stand de café oriental Café Elbey.

## Installation

```bash
npm install
```

## Lancer en développement

```bash
npm run dev
```

## Build de production

```bash
npm run build
```
Les fichiers finaux seront générés dans le dossier `dist/`.

## Structure

- `src/App.jsx` — page principale (hero, créateur, galerie défilante, services, contact)
- `src/index.css` — design system (couleurs, typographies, animations)
- `src/assets/` — images du stand (cover, founder, gallery-1 à gallery-6)
